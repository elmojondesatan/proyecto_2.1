Sistema de codigos abiertos

Se denomina código abierto a un modelo de producción descentralizada que permite que cualquier persona modifique y comparta tecnología porque su diseño es accesible de manera pública.

Sistma de codigo cerrado

En informática un programa es de código cerrado cuando el código fuente no se encuentra disponible para cualquier usuario, es decir no se hace público. Se le llama así en contraposición al código abierto.
